En esta carpeta se encuentra el archivo ZIP con los archivos de la calculadora básica con interfaz gráfica, por favor siga los siguientes pasos:
  1. Descargar los archivos
  2. Descomprimir el archivo zip
  3. Abrir el archivo "server.py"
  4. Ejecutarlo para así abrir el server
  5. Acceder a la siguiente IP "http://localhost:8000"
  6. Realizar las operaciones necesarias
